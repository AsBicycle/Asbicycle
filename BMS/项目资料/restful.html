/users

使用APP租用自行车的用户

/users/{userId} get put

get

获取当前用户的基本信息，不包括租车历史记录和充值信息。基本信息内容待定。

put

更新当前用户的基本信息

get /users/{userId}

获取当前用户的基本信息，不包括租车历史记录和充值信息。基本信息内容待定。

  • Response

  • userId: required (string)

HTTP status code 200

  "wechatAccount": "wechat_user",
 },

/trails

用户的历史租车记录，每条记录是一个租车骑行线路。

/trails get post

get

分页获取用户租车记录列表，按照时间倒序排列。

post

当用户租车成功后，生成一条新的线路信息。

get /trails

分页获取用户租车记录列表，按照时间倒序排列。

  • Request

Query Parameters

  • userId: (string)

    根据用户id查询租车记录，用户只能查看自己的记录

    Example:

    45453452345
  • bicycleId: (string)

    根据自行车id查询租车记录，保留功能，当前可能不实现

    Example:

    4567

post /trails
